# Good to Know in Jenkins
Jenkins Document : https://www.jenkins.io/doc/

Jenkins open on port 8080

## Directory
- /var/lib/jenkins : jenkins home directory
- /var/log : jenkins log
- /usr/lib/jvm : java home path (for manual install some tool like JDK)
