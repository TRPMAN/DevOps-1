## Important Directory
- /root, /home/username : Home Direc
- /bin, /usr/bin, /usr/local/bin : User Exe
- /sbin, /usr/bin, /usr/local/sbin : Sys Exe
- /media/mnt : Other Mountpoints
- /etc : Config
- /tmp : Temporary
- /boot : Kernal&Bootloader
- /boot : Server Data
- /proc, /sys : Sys Info
- /lib, /usr/lib, /usr/local/lib : Shared Lib
