## VIM Editor
- vim : edit file
- insert (button) : insert mode
# Command Mode
- ESC : Go in to command mode
- gg : go top
- G : go end
- u : undo
- U : undo Entire Line
- Ctrl + R : Redo Changes
- yy, (n)yy : copy line
- p : Paste line
- dd, (n)dd : delete line
- / : search word
# Extend Mode
- ESC : Go in to Extend Mode
- :w : save
- :q : quit
- :wq! : force
- :se nu : set line num
- :se nonu : remove line num
- :%s/(word replaced)/(replace word) : replace
  - --> /g(in the end) : All word
