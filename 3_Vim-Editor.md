## VIM Editor
edit file               : vim
insert mode             : insert (button)
# Command Mode
Go in to command mode   : ESC
go top                  : gg
go end                  : G
undo                    : u
undo Entire Line        : U
Redo Changes            : Ctrl + R
copy line               : yy, (n)yy
Paste line              : p
delete line             : dd, (n)dd
search word             : /
# Extend Mode
Go in to Extend Mode    : ESC
save                    : :w
quit                    : :q
force                   : :wq!
set line num            : :se nu
remove line num         : :se nonu
replace                 : :%s/(word replaced)/(replace word)
  --> All word          : /g(in the end)
